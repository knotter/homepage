A ordered list example

My preparation list for application to university:
  • 1. TOEFL test
  • 2. SAT
  • 3.Enrollment test
    • 1. Japanese
    • 2. Mathmatics
    • 3. English
    • 4. physics
    • 5. chemistry
  • 4.GPA

The instance above reveals that the command "ul" is used to incite a list, and of course "/ul" is used to end this list, and each "li" represents one particular content in the list. Though, when we want to set a child list, we view the child list as a content in the original list, do the same thing as mentioned above, and finally enclose the whole content with "/li", as can be showcased in the instance 3, Enrollment test. Be aware that when we want to compile a semantic tag to explain something in the list, we should let it enclosed by "li"and "/li".

Another example:

Oreo eating procedure:
  1. Open the box
  2. Take out cookies
  3. Make a Double Oreo
    1. Peel off the top part
    2. Place another cookie in the middle
    3. Put back the top part
  4. Enjoy!

We can see that when we use "ol" to incite a list, it automatically place our contents in a number sequence.
