Anatomy of a CSS Rule

Subheading 1

CSS works by associating rules with HTML elements. These rules govern how the content of specified elements should be displayes. A CSS rule contains two parts: a selector and a declaration.

Subheading 2

We use "style" to set a column of CSS rule to govern how those contents should be deployed. "color" determines color, "text-align" tells the contents where to be displayed, "font-size" adjust the size of each word.
