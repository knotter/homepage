Internal Links

We can link to a file in the same directory as this HTML like this: ***"a": Linking to a file in the same directory as this file.(list.html)
DIV linking to a file in the same directory(list.html)

Now, we learn that when we use command "a" and "href" only, we can target links to a file in the same directory as this file, which are called internal links.

External links

We can link to a designated website via this file like this: A link to bilibili.

We copy a specific address of a website and paste it after the "href" command to create a hyperlink to that website. Besides, the command "target" is use to create a new browser in case of the moment we still have something to do with present browser. By the way, "target" should be enclosed by <a>.
