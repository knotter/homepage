Box Model

Hints for Padding: We need 4 numbers representing width and heights to adjust the size of the padding(box). Its sequencial order symbolize the real box size data in clockwise order. E.G: the first 5 represents the upper height. Subsequently, the next 5 represents the right width.

Hints for box-sizing: it does not conform to the inheritance rule, which means it cannot apply to its child tag. However, we can have a star selector to amend this problem.

The rule for margin: When there are two elements adjacent with each other, the distance between them will be the sum of their margin. Instead, when paralleling, the distance between them will be the longer margin.
